Introduccion a Server App For Windows

Version: 1.0 - Beta 1


Esta version que contiene?

Esta es la primera versión de Server App For Windows. Esta version está en fase beta por lo que puede estar propenso a errores. Server App For Windows esta destinada al control de servidores de Minecraft Bedrock como saber su:

  • Versión
  • Edición
  • IP
  • Puerto
Ademas podemos medir el consumo de recursos basado en el administrador de tareas de windows como:
  • Uso de CPU
  • Uso de RAM
  • Uso de Red
De esta forma se puede manipular el rendimiento de la app. Segun cumplan los requsitos


Requisitos de instalación:

  • CPU: Intel® Core® I3 2° Generacion o Ryzen 3 2000 Series o superior
  • RAM: 2GB/3GB o superior
  • Almacenamiento: 1GB/2GB o superior
  • Red: Conectividad Minima de 5 Mbps para una experiencia fluida o superior
  • OS: Windows 10 1703 o Windows Server 2016 o superior
Estos requsitos son aplicados para que tengas una experiencia fluida y sin complicaciones

Instalación

  • Abres el archivo que se llama "ServerAppBeta1Installer.exe" No Se requieren permisos de administrador
  • Pulsas el boton "examinar" y seleccionas tu ruta de destino
  • Pulsa el boton "Extraer" y espera un momento
  • Va y buscas el directorio donde extrajo el paquete debe aparecer algo asi
    |
    + [Carpeta] dist
    + [Carpeta] code
    + [Archivo] NodeRunner.cmd
  • En la barra donde esta el directorio tocas el espacio vacio y escribes: NodeRunner -App esto va a empezar a ejecutar la app con normalidad
  • Seleccionas el directorio donde tengas el archivo bedrock_server.exe

Uso

  • En la pestaña principal el boton Iniciar Permite ejecutar el servidor en local
  • En la pestaña principal el boton Detener Permite Detener el servidor en local
  • En la pestaña principal el boton Reiniciar Permite detener y volver a ejecutar el servidor en local
  • En la pestaña de archivos tenemos las mismas funcionalidades que en el explorador de archivos
  • Pero recuerda el explorador de archivos SOLO ABRE ARCHIVOS EN FORMATO .properties .txt .json .exe .zip .tar.gz y carpetas. Incluye un editor de texto integrado para los archivos .properties .txt .json

Restablecer App

  • Ve al directorio de la instalación
  • En la barra donde esta el directorio tocas el espacio vacio y escribes: NodeRunner -AppRes
  • En la ventana que se abrio pulsa el boton "Restablecer Ahora" esto abrira un cuadro de dialogo
  • Pulsa en "Yes" en el cuadro de dialogo y esperas un momento
  • Pulsa "Ok" y luego cierra la ventana
